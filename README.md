## English

### âœ¨ Introduction

**Clean PigGo** is an AI-powered online image watermark removal tool. Using the advanced [LaMa (Large Mask Inpainting)](https://github.com/advimman/lama) deep learning model, it can intelligently remove watermarks, text, and unwanted objects from images.

ğŸŒŸ **Key Highlight: All processing is done entirely in your browser locally. Images are never uploaded to any server, fully protecting your privacy!**

### ğŸ¯ Live Demo

Visit **[clean.picgo.studio](https://clean.picgo.studio)** to try it now!

### ğŸš€ Features

| Feature | Description |
|---------|-------------|
| ğŸ¨ **Smart Erase** | Select the area to remove with the eraser tool, and AI will automatically fill in the background |
| âš¡ **Real-time Processing** | WebGL-accelerated ONNX Runtime enables processing directly in the browser |
| ğŸ”’ **Privacy Safe** | All processing is done locally, images are never uploaded to any server |
| ğŸ­ **Mask Technology** | Precise mask generation ensures only selected areas are processed |
| ğŸ‘ï¸ **Real-time Comparison** | Compare before and after effects with a side-by-side slider |
| ğŸ“¥ **HD Export** | Export results at original resolution, maintaining image quality |

### ğŸ“– How to Use

1. **Upload Image** - Drag and drop or click to upload an image (supports JPG, PNG, WebP)
2. **Select Area** - Use the eraser tool to paint over the watermark area
3. **AI Processing** - LaMa model automatically analyzes and fills the selected area
4. **Download Result** - Preview the result and download the processed image
